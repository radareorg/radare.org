iaito

iaito is the official graphical interface of radare2. It's the continuation of Cutter for radare2 after the Rizin fork.

Runs on Linux, macOS and Windows, it's written in C++ and uses the Qt framework.

For more information please checkout our Github project page. Alternatively you may want to Download the release builds.
